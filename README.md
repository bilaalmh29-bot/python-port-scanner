# Python Port Scanner

A simple TCP port scanner written in Python.

## Features
- Scans ports 1 to 1024
- Accepts IP addresses or hostnames
- Reports open ports
- Handles common errors gracefully

## Requirements
- Python 3.x

## Usage

```bash
python port_scanner.py
```

Enter a target such as:
- scanme.nmap.org
- 192.168.1.1

## Skills Demonstrated
- Python programming
- Socket programming
- TCP/IP networking
- Cybersecurity fundamentals
